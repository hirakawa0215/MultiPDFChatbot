# OpenAI-Chat-with-Multiple-PDFs-Streamlit-Web-App
Python Streamlit web app allowing the user to upload multiple files and then utilizing the OpenAI API GPT 3.5 Turbo language models, the user is able to have a conversation about the uploaded documents. The user is also allowed to specify the language model and the temperature/creativity of the model.
